# Carlos-Vinicius
Read
O programa criada tem por objetivo verificar qual a nota mínima necessária que um aluno do IME deve obter na Verificação final do semestre.
Deve - se inserir as médias de VE e VC, respectivamente.
Caso o aluno, por algum descuido, insira um grau indevido(ex: nota maior que 10 ou nota negativa), o programa informará o erro e solicitará a entrada do grau retificado.
